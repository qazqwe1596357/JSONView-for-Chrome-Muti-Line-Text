# JSONView-for-Chrome-Muti-Line-Text
JSONView for Chrome Muti Line Text

To install the plugin, only needed to
----
Download the "JSONView-MutiLineDisp.crx" file.
----

Addition Note:
----
安装方法

【Chrome】-【设置】-左侧【拓展程序】
 
 将 JSONView-MutiLineDisp.crx 文件拖拽到 Chrome 中，
 
 勾选【开发者模式】
 
 弹出对话框选择确认即可完成安装。
